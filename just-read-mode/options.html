Keyboard Shortcuts

Default keyboard shortcuts:

  • Ctrl+Shift+R (Windows/Linux) or Cmd+Shift+R (Mac) - Toggle reading mode
  • ESC - Exit reading mode
  • Ctrl++ or Ctrl+- - Adjust font size
  • Ctrl+D - Toggle dark mode

Note: To change keyboard shortcuts, visit chrome://extensions/shortcuts
