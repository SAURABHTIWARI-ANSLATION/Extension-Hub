## Installation

### For Development:
1. Download or clone this repository
2. Open Chrome/Edge and go to `chrome://extensions/`
3. Enable "Developer mode" (toggle in top right)
4. Click "Load unpacked"
5. Select the `spin-the-wheel-extension` folder

### Files Included:
- `manifest.json` - Extension configuration
- `popup.html` - Main popup interface
- `popup.js` - Popup functionality
- `popup.css` - Popup styling
- `wheel.html` - Fullscreen wheel page
- `wheel.js` - Wheel animation and logic
- `wheel.css` - Wheel styling
- `background.js` - Background service worker
- `icons/` - Extension icons
- `README.md` - This file

## How to Use

1. Click the extension icon in your browser toolbar
2. Add options you want to include on the wheel
3. Customize colors and spin duration
4. Click "Spin The Wheel!" to spin in popup
5. Or click "Open Fullscreen Wheel" for a bigger experience
6. View your spin history in the popup

## Customization

- Add/remove options as needed
- Choose from preset colors or modify in code
- Adjust spin speed and duration
- Wheel automatically saves your preferences

## Browser Support

- Chrome 88+
- Edge 88+
- Other Chromium-based browsers
