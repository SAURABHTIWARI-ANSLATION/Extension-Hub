Privacy Policy for API Tester Pro

Introduction

This Privacy Policy describes how API Tester Pro collects, uses, and discloses information, and what choices you have with respect to the information.

Data Collection and Usage

We do not collect personal data.

API Tester Pro is a tool that allows you to test APIs directly from your browser.

API Requests

  • The extension executes API requests (GET, POST, etc.) that you initiate.
  • Request headers, bodies, and responses are handled locally within the extension to display the results.
  • We do not intercept, collect, or store your API data on our servers.
  • The data is only transmitted to the API endpoints you specify.

Permissions Usage

The extension requests the following permissions for functionality purposes only:

  • storage: Used to save your request history and saved collections locally on your device.
  • <all_urls>: Required to allow the extension to permit cross-origin requests (CORS) to any API endpoint you wish to test, overcoming standard browser restrictions for testing purposes.

Third-Party Services

This extension does not use any third-party analytics or tracking services.

Changes to This Policy

We may update our Privacy Policy from time to time. Thus, you are advised to review this page periodically for any changes.

Contact Us

If you have any questions or suggestions about our Privacy Policy, do not hesitate to contact us.
